## Resumen

`ng-hub-ui-portal` permite renderizar componentes y plantillas en contenedores dinámicos del DOM, ideal para overlays, paneles, drawers y contenido desacoplado del árbol visual principal.

## Instalación

```bash
npm install ng-hub-ui-portal
```

## Uso rápido

```typescript
import { HubPortal } from 'ng-hub-ui-portal';
```

## Características

- Renderizado dinámico de componentes y plantillas.
- Soporte para múltiples contenedores.
- Gestión de foco y accesibilidad.
- Integración útil para overlays y sistemas de popups.
